\# Mini Platformer



간단한 2D 플랫폼 게임 미니 프로젝트입니다.



\## Project Info

\- Engine: Unity

\- Language: C#

\- Period: 2025.12 ~ 2026.01

\- Platform: PC



\## Features

\- Player movement

\- Jump / Double Jump

\- Basic collision



\## Notes

\- 개인 학습용 프로젝트

\- Git 사용 연습 목적
